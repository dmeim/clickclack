# ClickClack

A minimalistic typing test application inspired by Monkeytype.

## Features

- **Clean Interface**: Distraction-free typing environment.
- **Customizable**: Change themes, fonts, and more.
- **Multiple Modes**: Time, Words, Quotes, and Zen mode.
- **Statistics**: Track your WPM, accuracy, and consistency.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
